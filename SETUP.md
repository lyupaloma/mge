# Запуск проекта MGE DNA Landing

## 1. Установить Node.js

Скачай с nodejs.org версию LTS (20+) и установи. После установки открой новый терминал.

## 2. Установить зависимости

```bash
cd "C:\Users\Lyubov\Desktop\промт\AI-Workspace\projects\MGE\landing"
npm install
```

## 3. Запустить dev-сервер

```bash
npm run dev
```

Открой http://localhost:3000 — автоматически редиректнет на /ru/

## Языки:
- http://localhost:3000/ru  — Русский
- http://localhost:3000/kz  — Казахский  
- http://localhost:3000/zh  — Китайский

## Сборка для продакшна:

```bash
npm run build
npm run start
```
